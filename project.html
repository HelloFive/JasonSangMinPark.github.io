Breakout

Implemented a game Breakout in Java with swing and awt. This game handles real-time drawing and animation of simple graphics. Some special features (cheat key) are user can control size of paddle and ball, speed of ball, skip to any level 1-10.

Doodle

Implemented a painting program Doodle in Java with swing and awt. This painting program lets you paint with your chosen color and thickness of stroke. It is represented with respect to Graphical User Interface. Drag downward to decrease thickness of stroke, drag upward to increase the thickness of stroke. Double click to choose your color. Also, you can playback with history of your drawings with slider and go back to the stateby controlling the slider. Your drawing can be saved and loaded as a PNG file.
